# [BATDR] Ink Demon Timer Mod

## Installation
1. Install BepInEx into your game folder.
2. Download `InkDemonTimer.dll` from the Releases Tab and put it into `BepInEx/plugins`.

## Usage
You'll see a timer on the top of the screen when the Ink Demon is active.  
It will start blinking in the last 5s as a warning and will not be visible when he isn't active.

You can toggle the timer with `F9`.
